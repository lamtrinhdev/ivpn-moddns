### Development Certificates

This directory contains certificates necessary for local development and testing.


1. `private_key.pem` and `certificate.pem` are used in API unit tests and integration tests (mobileconfig generation).
2. `ivpndns.com+4.pem` and `ivpndns.com+4-key.pem` are used for local development and in integration tests.
3. `mkcert_development_CA_307611231582065277882115426409270736451.crt` is a root certificate used in integration tests (both locally and in Github workflow.)
